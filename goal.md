## Goal

I want to create a firefox extention that can read the page im on, and read it out loud to me like a teacher would teach me.

connect it with a gemini api to get the summary of the page and read it out loud to me.

## Features

- Read the page im on out loud to me.
- Get the summary of the page and read it out loud to me.
- human sounding voice.
- intergrate with gemini so i can ask questions about the page.
